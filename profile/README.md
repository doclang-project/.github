# Welcome to the DocLang, **the** AI ISO format for Unstructured content

This is the GitHub organization DocLang ISO standard. We like to get continuous feedback from the community!

The founding members of the standard are:

1. IBM
2. ABBYY
3. RedHat
4. HumanSignal
5. Nvidia

We are open to accept invitations to join the DocLang governing team.
